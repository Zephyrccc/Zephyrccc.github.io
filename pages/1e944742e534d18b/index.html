# 思路分析

本题比较简单，一开始将后门函数的地址就打印了出来，可以接收该地址，再通过gets函数溢出将返回地址修改为该值即可获得flag。

# 确定偏移量

使用cyclic，简单测试一下即可得知本题偏移量为0x40+8=72
